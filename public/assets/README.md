# Playbook downloadable assets

Files in this folder are linked from **Module 15 — Downloadable Assets** in each vertical playbook. Keep filenames stable so training links do not break; replace contents when marketing ships a new kit.

| File | Purpose |
|------|---------|
| `logo.png` | Raster logo (slides, docs, social) |
| `logo.svg` | Vector logo (web, proposals) |
| `email-banner.png` | Wide banner for email campaigns |
| `email-footer.html` | HTML snippet for email signatures |

Placeholders are included for local development. Swap in production-ready assets before go-live.
